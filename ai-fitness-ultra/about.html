🚀 About This Project

This project is a next-generation AI-inspired Fitness Planner designed using only frontend technologies like HTML, CSS, and JavaScript.

It simulates intelligent decision-making by generating personalized workout plans based on user input, creating the feel of a real AI system without using any backend.

The application includes modern UI features such as glassmorphism design, animated particle background, voice input functionality, and an interactive dashboard.

This project demonstrates strong skills in frontend development, responsive design, and creative UI design.
